====
Mask
====

.. only:: html

   :Extension key:
      mask

   :Version:
      |release|

   :Language:
      en

   :Description:
      Mask is a content element builder that generates TypoScript, TSconfig and
      TCA on the fly. Build your content elements in a user-friendly backend
      module via drag and drop. The definitions are stored in json files.

   :Keywords:
      content elements, backend layouts, TCA, builder, generator
